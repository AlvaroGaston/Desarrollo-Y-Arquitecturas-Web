# Web de Noticias de Baloncesto: NBA y Euroliga

Este proyecto es un portal de noticias deportivo desarrollado puramente con HTML y CSS. Consta de una página principal (`index.html`) y secciones dedicadas a la NBA y la Euroliga, estructuradas con un diseño moderno y responsivo utilizando Flexbox.

## Fuentes Originales de Noticias

A continuación, se detallan las URLs de los sitios web de noticias reales utilizados como referencia y enlazados en el proyecto para que los tutores puedan comparar el contenido:

### Sección NBA (`nba.html`)
* **NBA Oficial (Resultados y estadísticas):** [https://www.nba.com](https://www.nba.com)
* **ESPN (Rumores y mercado):** [https://www.espn.com/nba/](https://www.espn.com/nba/)
* **Bleacher Report (Análisis profundo):** [https://www.bleacherreport.com/nba](https://www.bleacherreport.com/nba)

### Sección Euroliga (`euroliga.html`)
* **Euroleague Basketball (Sitio Oficial y Clasificación):** [https://www.euroleaguebasketball.net/euroleague/](https://www.euroleaguebasketball.net/euroleague/)
* **BasketNews (Actualidad y MVP):** [https://basketnews.com](https://basketnews.com)
* **Diario Marca (Sección Euroliga):** [https://www.marca.com/baloncesto/euroliga.html](https://www.marca.com/baloncesto/euroliga.html)
